vSOC / IT-OT

SOC-Adeodato

Solución vSOC (SOC gestionado) IT-OT para PYME e industria. Detección en tiempo real con Wazuh, contención automática con Shuffle, investigación en DFIR-IRIS y malla de red segura con Tailscale.

Wazuh Shuffle SOAR DFIR-IRIS Tailscale
GitHub
Android App

AdeoSOC

Aplicación móvil nativa en Kotlin para la gestión y recepción de alertas prioritarias del vSOC. Permite contención remota y monitoreo en tiempo real desde dispositivos móviles.

Kotlin Android SDK Push Notifications vSOC Mobile
GitHub
Android / DoH

AdeoShield

Sistema de control parental y filtrado de contenido para Android basado en DNS-over-HTTPS (DoH). Protege la navegación cifrando consultas y bloqueando dominios maliciosos o no deseados.

Kotlin DNS-over-HTTPS Android Security GPL-3.0
GitHub
Python Tool

ARGOS

Herramienta en Python orientada a la inteligencia de amenazas (CTI) y automatización de recolección de indicadores de compromiso (IoCs) para fortalecer la postura defensiva del SOC.

Python 3 CTI Automation GPL-3.0
GitHub
